How to get your Salesforce Consumer Key:

  1. Go to Salesforce
  2. Navigate to Setup â†’ App Manager
  3. Find your "Astro" app or create a new Connected App
  4. Enable OAuth Settings
  5. Add these callback URLs: astro://oauth/salesforce/callback and https://localhost:3000/oauth/salesforce/callback
  6. Copy the Consumer Key

How to get your Slack Client ID:

  1. Go to Slack API
  2. Create a new app or select your existing "Astro" app
  3. Go to Basic Information â†’ App Credentials
  4. Copy the Client ID
  5. Add these redirect URLs: astro://oauth/slack/callback and https://localhost:3000/oauth/slack/callback

How to get your Deepgram API Key:

  1. Go to Deepgram Console
  2. Sign up or log in
  3. Go to API Keys section
  4. Create a new API key
  5. Copy the API key
